Installing PIP to manage Python packages
pip3 - command
pip3 install Flask (Installs python Flask)
Flask: Installs to /home/dev/.local/bin   (not on PATH)

"export FLASK_APP=name.py" allows "flask run" command to start webserver


//Debugging
"export FLASK_DEBUG=1" Turns on debug mode
